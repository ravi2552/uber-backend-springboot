# Uber Clone Backend

Backend system for an Uber-like ride booking application built using Spring Boot.

## Tech Stack

- Java 21
- Spring Boot
- Spring Security
- JWT Authentication
- PostgreSQL
- Maven

## Features

- User Authentication (Signup/Login)
- Rider and Driver management
- Ride request system
- Ride status tracking

## Project Structure

src
├ controllers
├ services
├ repositories
├ entities
├ security
└ config
